# Lesson Platform

A deployed, online lesson platform where each lesson combines a recorded video (from the
HGL-Console studio), notes, and interactive questions — all ingested from JSON and
organised as subject → topic → topic area → lesson.

## Getting started

There is no build step yet (the stack decision is pending). Open
[`index.html`](index.html) directly in a browser to load the scaffold page.

## Project anchor

Read [`CLAUDE.md`](CLAUDE.md) first — it is the durable source of truth for the project
summary, workflow rules, engineering lessons, the design system, and the JSON contracts.
